ARROW-API

Last updated: 2022-12-08

ARROW-API enables developers to build automated trading solutions on the Algomojo infrastructure. Our goal is to provide a comprehensive API set that enables traders to convert their trading ideas into a fully functional algorithmic trading systems and analytics.

Here are some of the things you can build with Algomojo

1)Want to build multi broker/multi exchange algo trading solutions? We have built everything you need.
2)Want to send Orders from Multiple Platforms like Amibroker, Tradingview, Excel, Python and C-Sharp based trading platforms? We support all in one multi-broker bridge
3) Want to built your own Algo Trading Framework that we don't officially support yet? Everything you need is here. You can use the Algomojo to build customized trading application right from scratch.

ARROW API URL

https://amapi.algomojo.com/v1/
