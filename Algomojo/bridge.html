ARROW-BRIDGE

Last updated: 2022-12-08

ARROW-Bridge is a connectivity layer between your trading platform and your broker. At Algomojo the bridge is designed to adapt multiple broker and can send/modify /cancel orders simultaneously using the Multi Broker Bridge Execution module.

ARROW-Bridge Installation

How to Install Algomojo Amibroker Bridge?

Step 1: Download bridge from the below link

Multibroker Execution Module
Step 2: Run AMBRIDGE-Setup.msi file in the system

Step 3: Click NEXT -> NEXT while installation

Step 4: Then press the Close button to finish the installation

Step 5: Download, Right Click and Run As Administrator the batch file 32-bit or 64-bit based on your Amibroker Version or Metatrader version
